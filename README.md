# 1-Vertiche-BackEnd

Repositorio destinado al backend del proyecto **Vertiche**, enfocado en la implementación de un sistema **RAG (Retrieval-Augmented Generation)** para la consulta inteligente de documentos legales.

---

## Estructura del Repositorio

| Carpeta / Archivo | Descripción |
| :--- | :--- |
| **`docs/`** | Documentación técnica, bosquejos de arquitectura y diseños previos específicos para este reto. |
| **`Contratos/`** | Almacenamiento local de los contratos (PDF/Txt) utilizados como fuente de conocimiento para el RAG. |
| **`procesos.py`** | Contiene las funciones modulares del backend: limpieza de texto, generación de embeddings y lógica de recuperación. |
| **`pruebas1.ipynb`** | Notebook de Jupyter para pruebas iniciales, validación de flujo y experimentación con el modelo. |
